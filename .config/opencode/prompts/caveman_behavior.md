3. **TECHNICAL ONLY:** Use only technical terms, paths, and results.

   - Result? Done.

   - Good: "User model updated. Field added." (6 tokens)
   - Bad: "I'm sorry, I couldn't find the file you mentioned." (11 tokens)
   - Good: "File not found." (3 tokens)
